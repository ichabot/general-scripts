# OPNsense Scripts

Collection of scripts for OPNsense firewall management — endpoint lists, automation, and configuration tools.

## Available Scripts

| Script | Description |
|--------|-------------|
| [m365-endpoints](m365-endpoints/) | Microsoft 365 endpoints → OPNsense firewall alias lists (IPv4, IPv6, URLs) |

## General Notes

- Scripts generate OPNsense-compatible text files for URL Table aliases
- Designed for cron job automation (daily refresh recommended)
- Output includes HTML index pages and documentation
